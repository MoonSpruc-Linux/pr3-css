# Practice 2 HTML

### Що стилізовано:

- **Header** та **footer** (фон, кольори, вирівнювання)
- **Навігація** (Flexbox, стани hover та active)
- **Контейнер та секції** (.container, .card, section)
- **Тексти**: заголовки, абзаци, списки
- **Зображення, відео та аудіо** (адаптивна ширина, рамки, тіні)
- **Таблиці** (підпис, відступи, чергування кольорів рядків)
- **Форма** (поля, кнопки, стани :hover та :focus)
- **Адаптивність** для екранів ≤768px через media query
